About

Roblox scripter with a focus on clean, modular code

Five years ago, I wrote my first line of Luau. It wasn't pretty, but it sparked something. Since then, I've evolved from following basic tutorials to architecting complete game systems in Roblox Studio.

I've learned that complexity isn't always the answer. Whether it's a datastore, combat system, or NPC AI, I write efficient, modular scripts that don't tank your server and are easy to maintain long after I'm gone.

When we work together, I keep things straightforward: fast responses, no mid-project ghosting, and scripts that actually work with your specific setup before delivery.
